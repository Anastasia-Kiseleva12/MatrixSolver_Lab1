# MatrixSolver_Lab1

Лабораторная работа №1 по дисциплине «Численные методы».

Тема: прямые методы решения систем линейных алгебраических уравнений (СЛАУ).

## Реализованные возможности

### Решение СЛАУ

В проекте реализованы следующие методы решения системы $Ax=b$:

1. Простой метод Гаусса
2. Метод Гаусса с выбором ведущего элемента по столбцу
3. Метод Гаусса с выбором ведущего элемента по всей матрице
4. Решение с помощью библиотеки MathNet.Numerics

Для каждого метода вычисляется решение системы и норма невязки

$$
|Ax-b|_\infty
$$

---

### Обратная матрица

Обратная матрица вычисляется двумя способами:

* собственной реализацией через решение систем

$$
Ax=e_i
$$

* с использованием библиотеки MathNet.Numerics

Полученные результаты сравниваются.

---

### Число обусловленности

Для матрицы вычисляются числа обусловленности:

$$
\operatorname{cond}_1(A), \qquad
\operatorname{cond}*2(A), \qquad
\operatorname{cond}*\infty(A)
$$

Это позволяет оценить чувствительность решения к погрешностям исходных данных.

---

### Дополнительное задание

Реализирована генерация плохо обусловленной системы, для которой:

* простой метод Гаусса даёт менее устойчивый результат;
* методы с выбором ведущего элемента работают лучше.

Также реализован эксперимент с возмущениями:

* к матрице $A$ и вектору $b$ добавляются малые случайные возмущения;
* решается новая система;
* вычисляются относительные изменения:

$$
\frac{|\Delta A|*\infty}{|A|*\infty}, \qquad
\frac{|\Delta b|*\infty}{|b|*\infty}, \qquad
\frac{|\Delta x|*\infty}{|x|*\infty}
$$

Это позволяет экспериментально подтвердить плохую обусловленность системы.
